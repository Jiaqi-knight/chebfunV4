The Chebfun Toolbox

Release 2008a. Copyright © 2002-2008 by the Chebfun Team. Please see the software license for usage restrictions.

The Chebfun Toolbox provides many familiar MATLAB commands with appropriate interpretations for functions of one variable. The aim of the chebfun system is to "feel symbolic but run at the speed of numerics". More precisely our vision is to achieve for functions what floating-point arithmetic achieves for numbers: rapid computation in which each successive operation is carried out exactly apart from a rounding error that is very small in relative terms.

The Chebfun Toolbox is not associated in any way with The MathWorks, Inc., which produces MATLAB.
